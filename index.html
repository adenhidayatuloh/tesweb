My Latest Project

E-Smartcare (Stunting detection application)

E-Smartcare is a stunting detection application project, based on Android. The technology used is flutter on mobile, and golang on the server side. In this application there are features such as stunting detection based on photos, to AI chat bots using the concept of information retrieval.

Sehati Sehat (Health Check Application)

Sehati Sehat is a health application designed to help patients, admins, and health workers in various activities related to disease detection, health monitoring, emergency response services, and health data management. This software is designed to detect diseases in humans using a combination of expert system technology and machine learning.
